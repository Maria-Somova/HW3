Автор ФИО: Сомова Мария Александровнa, Группа: ББИ2506

Тематика: Проект симулирует работу аналитического модуля для студии маникюра.

Структура генирируемых данных:
* "manicure_type" - тип услуги
* "price" - цена услуги
* "time" - время оказания услуги
* "master_level" - уровень мастера

Структура проекта:
.
├── data/                    
├── generator/               
│   ├── Dockerfile
│   └── generate.py
├── reporter/               
│   ├── Dockerfile
│   ├── package.json
│   └── report.js
└── run.sh                   

Инструкция по запуску:
0. Если запускаете скрипт впервые, выдайте ему права на использование в терминале: chmod +x run.sh
1. Сборка образов генератора данных: ./run.sh build_generator
2. Запуск генерации csv файлов: ./run.sh run_generator
3. Сборка образа аналитики: ./run.sh build_reporter
4. Запуск аналитики: ./run.sh run_reporter

Дополнительные команды: 
* Просмотр структуры: ./run.sh structure
* Проверка видимости данных из контейнера генератора: ./run.sh inside_generator
* Проверка видимости данных из контейнера аналитики: ./run.sh inside_reporter
* Очистка данных: ./run.sh clear_data
* Локальная генерация: ./run.sh create_local_data
